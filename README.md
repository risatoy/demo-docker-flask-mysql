## Installation
1. clone this repo

2. change mysql_password to your mysql password inside app/templates/db.yml

3. change MYSQL_ROOT_PASSWORD to your mysql password inside docker-compose.yml

4. `docker-compose build`

5. `docker-compose up`

6. localhost:5001
